## 📖 README: Como Rodar 🚀

Aqui está o guia rápido para colocar seu jogo para funcionar:

### 🛠️ Pré-requisitos

* Um compilador de C (como o **GCC**) instalado.
* Ou use um editor online como o **GDB Online**.

### 🏃 Passos para Executar

1. **Copie o código** acima em um arquivo chamado `batalha-naval.c`.
2. **Abra o terminal** na pasta do arquivo.
3. **Compile o programa**:
```bash
gcc batalha-naval.c -o batalha-naval

```

4. **Execute**:
```bash
./batalha-naval

```

### 🎮 O que você verá no console?

* `0`: Água 🌊 

* `3`: Navios posicionados 🚢 

* `5`: Áreas afetadas pelas Habilidades Especiais ⚡
